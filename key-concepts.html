Key Concepts

Short, beginner definitions you can read fast and remember:

Request
The message a browser sends to ask for something (a page, image, data, etc.).
Response
The server’s answer, including a status code and the content.
Status Code
A number that tells how the request went (200 OK, 404 Not Found, 500 Server Error).
Header
Extra info about the request or response (content type, language, cookies).
HTTPS
HTTP with encryption (TLS) for privacy and integrity.
HTTP/2 Multiplexing
Sending multiple things at the same time on one connection.
Cache
A saved copy of a file (like CSS or images) so the browser can reuse it quickly.
Method
The action in a request (for example, GET to fetch, POST to send data).
